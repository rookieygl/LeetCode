## 线程打印
创建3个线程，每个线程分别可打印A、B、C。请按顺序输出ABCABCABC重复10次
```java
public class Item extends Thread {
    private static int currentCount = 0;
    public Item(String name) {
        // 设置线程的名称
        this.setName(name);
    }

    @Override
    public void run() {
        while (currentCount < 30) {
            switch (currentCount % 3) {
                case 0:
                    if ("A".equals(getName())) {
                        System.out.print(getName());
                        currentCount++;
                    }
                    break;
                case 1:
                    if ("B".equals(getName())) {
                        System.out.print(getName());
                        currentCount++;
                    }
                    break;
                case 2:
                    if ("C".equals(getName())) {
                        System.out.print(getName());
                        currentCount++;
                    }
                    break;
            }
        }
    }

    public static void main(String[] args) {
        // 线程启动顺序无关
        new Item("A").start();
        new Item("B").start();
        new Item("C").start();
    }

}
```

## 约瑟夫环
```
// 标号1-n的n个人首尾相接，1到3报数，报到3的退出，求最后一个人的标号
public void cycle(int n, int callExit) {
    int[] arr = new int[n + 1];//报数从1开始所以数组大小为n+1
    int callCount = 0, exit = 0;
    int i;
    //记录位置
    for (i = 1; i <= n; i++) {
        arr[i] = i;
    }

    for (i = 1; ; i++) {
        if (i > n) {
            //循环
            i = i % n;
        }

        //退出的不再计数
        if (arr[i] > 0) {
            callCount++;
        }

        if (callExit == callCount && exit != n - 1) {
            //退出
            callCount = 0;
            arr[i] = 0;
            exit++;
        } else if (exit == n - 1) {
            //最后一人退出
            System.out.print(arr[i] + "");
            break;
        }
    }
}
```
